# SecureHER AI

SecureHER AI is an AI-powered confidential reporting platform designed to help women safely report harassment, discrimination, and unsafe situations.

## Features
- Anonymous complaint submission
- AI-based complaint classification
- Automated risk scoring
- Emergency case prioritization
- Evidence upload system
- Live location tracking
- Admin dashboard with analytics
- Complaint status tracking

## Tech Stack
- HTML
- CSS
- JavaScript
- Chart.js

## Purpose
This project was built for a hackathon to create a safer digital platform for reporting incidents while ensuring privacy and faster response.
